\# Real Time Chat App

A real-time chat application built using Node.js, Express, and Socket.IO.



\## Features

\- Real-time messaging

\- Chat history on page refresh

\- Multiple users support

\- Simple UI



\## Tech Stack

\## How to Run Locally

1\. Clone the repository: `git clone https://github.com/Himanshu-dimri97/real-time-chat-app.git`

2\. Navigate to the server folder: `cd real-time-chat-app/nodeServer`

3\. Install dependencies: `npm install` (this will install all required packages including express and socket.io)

4\. Start the server: `node server.js`

5\. Open in browser: `http://localhost:3000`
